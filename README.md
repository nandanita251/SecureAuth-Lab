## 🚀 Overview
SecureAuth Lab is a playground designed to visualize and test advanced authentication workflows. Moving beyond the standard 2FA, this lab implements a **3-factor defense mechanism** to explore how modern applications handle identity, session security, and context-based access.

## 🔑 Key Features
* **Layered Authentication:** Password (Knowledge) + TOTP (Possession) + Context (Inherence).
* **Security Sandboxing:** Real-time demonstration of how JWTs and password hashing behave under different conditions.
* **UI/UX Focus:** Clean, interactive interface built with React for a seamless security experience.

## 🛠 Tech Stack
| Component | Technology |
| :--- | :--- |
| **Backend** | Python (Flask), `flask-jwt-extended`, `pyotp` |
| **Frontend** | React, JavaScript |
| **Security** | `bcrypt` (Hashing), WebAuthn (Context) |
